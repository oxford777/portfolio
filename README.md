---

## ✨ À propos du projet

Ce portfolio a été conçu et développé en HTML, CSS et JavaScript dans le but de créer une vitrine professionnelle moderne, responsive et accessible.

Le projet met en avant mes compétences en responsive design et en développement front-end à travers une interface claire, fluide et adaptée à tous les écrans.

Une attention particulière a été portée à l’expérience utilisateur, à la structure du code, à l’accessibilité ainsi qu’aux bonnes pratiques SEO.
